# About

This repository contains source code for firmware update integration product
webpage

# Local development

## Environment setup

```bash
$ sudo apt-get install ruby-full
$ gem install bundler
$ cd docs && bundle install
```

## Build

```bash
$ cd docs
$ cd bundle exec jekyll build
```

## Preview

```bash
$ cd docs
$ cd bundle exec jekyll serve
```
